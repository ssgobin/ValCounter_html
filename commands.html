Comandos do Bot

Comandos disponíveis:

      !vitoria - Use este comando para registrar uma vitória. Exemplo: !vitoria
      !derrota - Use este comando para registrar uma derrota. Exemplo: !derrota
      !removervitoria - Use este comando para remover uma vitória. Exemplo: !removervitoria
      !removerderrota - Use este comando para remover uma derrota. Exemplo: !removerderrota
      !resetar - Reseta as vitórias e derrotas. Exemplo: !resetar
      !registrar - Use este comando para registrar seu canal e permitir que o bot funcione corretamente (apenas Moderadores). Exemplo: !registrar

Instruções:

Para usar os comandos, basta digitá-los no chat do seu canal. O bot registrará suas vitórias e derrotas e permitirá que você tenha um controle melhor sobre suas estatísticas!
