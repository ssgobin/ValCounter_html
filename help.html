Ajuda para você!

Parabéns, você adicionou seu canal para o ValCounter funcionar!

Para prosseguir, adicione o bot como um moderador do seu canal (fique tranquilo que ele não vai roubar seu canal rsrs) utilizando o comando "/mod valcounter".

Logo após, utilize o comando "!registrar"

Pronto! agora é só começar a usar o bot! Não se esqueça de adicionar o overlay para que funcione direitinho.

Para ver a lista de comandos, basta acessar a página de comandos no cabeçalho da página.
